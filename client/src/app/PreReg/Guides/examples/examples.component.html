The Engineering group is developing basic problem sets for demonstration and tutorials.

When you first receive an account, you are provided a set of tutorial 'projects' that you observe and modify as you wish to 'try out'.

Fear not: you are allowed to reload the projects if you mess them up :-)

A few tutorial problems:

  • Find the 10th Root of 25 billion; like a square root only harder.
  • The Traveling Saleman Problem - who must visit each of a number of cities in the most efficient order.
